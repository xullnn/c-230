Right click:

- make the menu
  - Show Details
  - Edit Todo
  - Delete Todo

- when right click
  - get the coordinates of the cursor
    - event.pageX and pageY
  - show the menu based on the position
  - when click on the 'delete' option
    - update current_id
    - show the prompt widget

- when click on anywhere other than the menu
  - dismiss the menu
